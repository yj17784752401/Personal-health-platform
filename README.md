# sportplay
基于vue+springBoot的个人运动平台 
框架 
后端：springBoot+mybatis+json+java 
前端：Vue + Element-ui + router + axios + less + less-loader
数据库：mysql
后端链接地址：https://github.com/yj17784752401/Personal-health-platform-end

npm install
